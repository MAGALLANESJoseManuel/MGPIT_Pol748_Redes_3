Diameter
¶

La sesin anterior presentamos el concepto de dimetro. Si calculamos la menor distancia entre todos los nodos, el dimetro es la mayor de ellas.

Networkx slo calcula dimetro en un grafo conectado, y la red tiene varios componentes, por ello esto no funcionara:

In [10]:
# nx.diameter(PeruNet_AdjMx)

Lo que queda, es calcular el dimetro del mayor componente de la red (que llamaremos: PeruNet_AdjMx_bigger):

In [11]:
PeruNet_AdjMx_nodesInBig = sorted(nx.connected_components(PeruNet_AdjMx), key=len, reverse=True)

PeruNet_AdjMx_bigger=PeruNet_AdjMx.subgraph(PeruNet_AdjMx_nodesInBig[0])

#diametro del componente mayor
nx.diameter(PeruNet_AdjMx_bigger)
Out[11]:
4

El dimetro se haya de entre los distancias mnimas entre todos los nodos, por lo que tambin podemos saber el promedio de todas esas distancias:

Link analysis
¶

En esta sesin presentamos dos mtricas ms:

  1. HITS
  2. PageRank

Ambos son algoritmos para detectar nodos relevantes en la red, y van ms all del simple hecho de tener muchas conexiones. Estos algoritmos estn planteados para redes dirigidas, e inspirados en cmo un buscador web sugiere otras pginas.

HITS
¶

El algoritmo HITS busca calificar un nodo si es buen HUB o buena AUTHORITY. Sers mejor HUB si envas link a buenos AUTHORITIES, y sers buen AUTHORITY si recibes links de buenos HUBS. De ahi que cada nodo tiene una puntuacin para cada caso. Si la red es NO dirigida, la puntuacin es la misma (no es tan til). La Figura 16 representa el concepto.

En Python puedes calcularlo as:

In [16]:

# dando formato

HITS_authorities=pd.Series(a_dict,name="HITS_authorities").round(4)/2
pd.concat([HITS_hubs,HITS_authorities],axis=1)

Connected components
¶

La sesin anterior definimos como "red conectada" a la red cuyos nodos pueden comunicarse unos a otros. Las redes dirigidas que cumplen ello son "fuertemente conectadas", y se les llama "dbilmente conectadas" si no se considera la direccin. Las NO dirigidas son conectadas si son un nico componente, cuando tienen otros componentes que satelitan, ya son no conectadas. La Figura 20 lo ejemplifica.
